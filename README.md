Onde acessar?
Após reiniciar, abra o navegador e use esta URL:

http://localhost:8080/swagger-ui/index.html

Nesta página, você verá o botão "Try it out". Clique nele, preencha o email e senha no JSON e clique em Execute. Ele vai disparar a requisição diretamente para o seu código no IntelliJ e mostrar a resposta na tela.
